Python全栈工程师课程大纲
Python full stack engineer syllabus
1、课程准备>了解课程和学习计算机基础
学习安排：1周1次作业
课程内容：课程目标、课程简介与工具、全栈的概念与未来、课程教学体系、学习最佳实践、计算机基础、打造编程环境
2、Python基础>从小白到掌握Python
学习安排：2周2次作业
课程内容：成长型项目：51备忘录v0.21、字符串与数字、基本数据结构dict，set，tuple、控制程序的每个角落、了解函数的来龙去脉、给你想要的正则表达式、面向对象大法好
3、Python进阶>让工作开始自动化
学习安排：3周3次作业
课程内容：管理目录与文件、处理任意格式的文本文件、熟练操作各种办公文件、给程序加点日志、怎么发邮件、做个小爬虫、很多时候，不用自己造轮子、迭代器，生成器，装饰器
4、web基础>为网络服务开发做好准备
学习安排：5周5次作业
用Python控制服务器、使用Python操作数据库、常用网络服务中间件、web应用的历史与特点、HTML5登上历史舞台、CSS3为你的网页增光添彩、JavaScript与ES6/7让网页动起来
5、web前端框架>给你的前端代码加点料
学习安排：4周4次作业
课程内容：年纪略大的jQuery、最受欢迎的Bootstrap4、工程化前端项目、流行框架：Vue
6、web后端框架>网络服务的基石
学习安排：2周2次作业
课程内容：全身肌肉的Django
7、web全栈开发>打造属于你的网络应用
学习安排：3周3次作业
课程内容：前后端分离简介、REST API的作用、跨域与安全、项目：51备忘录v1.0（Django + Vue）、项目：电子商城（Django + Vue）、项目：微信小程序、项目：差评分析平台
8、敏捷开发与架构>工作必备与进阶技能
学习安排：3周3次作业
课程内容：敏捷开发流程、使用git协作开发、持续集成与部署、微服务与架构、面试常见问题
9、Python未来>给你终身学习的技巧
学习安排：4周4次作业 课程内容：算法与设计模式、简历与面试、学习代码规范、全栈工程师的日常、人人都是产品经理、打造你的全球智囊团、人工智能入门实践
10、Python+人工智能>机器学习入门篇
学习安排：3周3次作业
课程内容：从机器学习基本算法开始逐渐到进阶内容，对于复杂的机器学习算法首先给出通俗解释并对原理进行推导解析，最后阐述应用领域与价值，对于每一个机器学习算法配套案例实战
