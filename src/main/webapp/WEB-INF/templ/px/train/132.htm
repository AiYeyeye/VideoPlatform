Windows企业全能工程师课程大纲

  • 第一模块：Windows Server 2012 R2 管理
    学习安排：11周1次综合实验

    测试内容：虚拟化迁移方案
    学习形式：录播视频+在线作业考核+助教一对一辅导答疑
    课程内容：全面覆盖Windows Server 20122系统中主要的角色和功能，从大规模批量的系统部署，到活动目录和组策略的集中管理和配置，再到流行的虚拟化 Hyper-V，从功能和应用场景开始，逐步深入到运维中的自动化，可用性，可靠性等关键特性，逐一详细详明和介绍。

  • 第二模块：System Center 2012 R2 管理
    学习安排：6周1次综合实验

    测试内容：System Center集成应用方案设计
    学习形式：录播视频+在线作业考核+助教一对一辅导答疑
    课程内容：全面覆盖 System Center 2012 R2 的产品线和功能，集中式的操作系统，应用程序，合规性管理，到系统监控预警，自动化，流程管理，虚拟化和私有云平台，集中灾难恢复等，从IT管理的角度，逐一展开，全面而深入。

  • 第三模块：PowerShell 自动化运维管理
    学习安排：1周1次综合实验

    测试内容：Exchange Lync SharePoint 集成方案设计
    学习形式：录播视频+在线作业考核+助教一对一辅导答疑
    课程内容：Powershell 是微软最新一代的自动化管理利器，我们将从 Powershell 的组件，对象，命令行开始，详细说明通过 Powershell 实现本地的，远程的自动化配置和管理，WMI 的信息收集，高度集成的函数和自定义的 Module，以及实现 DevOps 的 Powershell DSC 功能。

  • 第四模块：统一通信管理
    学习安排：4周1次综合实验

    测试内容：Powershell 信息自动化收集
    学习形式：录播视频+在线作业考核+助教一对一辅导答疑
    课程内容：Exchange，SharePoint和 Lync 在企业中占有绝对统治地位，通过这一个课程，我们将深入学习搭建丰富类型的协作平台，实现邮件，语音，视频，电话的通信，实现日历，日程安排和项目的管理。

  • 第五模块：SQL Server 2012 管理
    学习安排：2周1次综合实验

    测试内容：SQL Server AlwaysOn方案设计
    学习形式：录播视频+在线作业考核+助教一对一辅导答疑
    课程内容：深入的介绍数据库平台的构建，数据库的性能优化，安全管理，高可用性和可靠性设计，实现数据分析和报表服务。

  • 第六模块：微软公有云管理
    学习安排：1周1次综合实验

    测试内容：Azure与AD混合环境
    学习形式：录播视频+在线作业考核+助教一对一辅导答疑
    课程内容：微软因公有云而再次雄起，我们也将通过这一次课程深入了解 Azure 的虚拟机，存储，网站的运营和管理，实现云计算中的 saas，paas和Iaas 不同层面的公有云平台
