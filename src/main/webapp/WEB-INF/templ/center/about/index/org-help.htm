51CTO学院合作机构申请条件
51CTO学院欢迎IT业界同行加盟机构合作，更好的为学员服务，一起打造IT在线教育课堂，申请机构需要满足以下条件：
1. 有意向发展/已有线上教育服务的IT培训机构/公司
2. 申请机构必须在51CTO学院平台提供收费课程
3. 申请机构必须至少有1名讲师
合作机构申请方法
机构需在线提交申请：http://edu.51cto.com/partner/add.html ，务必留下联系人联系方式
机构合作的操作流程
在线申请机构合作->学院管理员收到申请后联系机构联系人确认合作细节->合作机构提供相关信息->上线机构及旗下讲师页面->双方签定合作协议->机构开始上传课程。
上线机构页面要求
1. 双方洽谈达成合作事宜
2. 机构提供51CTO学院需要的各项信息，51CTO学院签约机构信息表>>
3. 机构在其网站首页加51CTO学院友情链接（文字：51CTO学院 链接：http://edu.51cto.com/）
51CTO学院在机构页面也会加上机构网站链接、联系方式等信息，例如：

4. 51CTO学院上线机构页面，例如：http://edu.51cto.com/partner/view/id-8.html
机构账号要求
学院规定，合作机构至少要注册一个机构51CTO账号、一个讲师51CTO账号。
机构账号：用于统一管理，提取现金收入等，机构下的个人讲师账号没有体现权限。
讲师账号（至少一个，可以用多个）：用于创建课程上传视频、回答用户问题等。
一个机构可以有多名讲师吗？如何新增新讲师？
学院规定，一个机构可以有多名讲师，每个讲师需要有一个51CTO用户名。
讲师可以随时增加，增加方法：根据51CTO学院签约机构信息表要求，提供讲师的相关信息并联系管理员元子增加即可（联系QQ：1844834113）
机构提现方法及注意事项
学院规定当机构账户金币满2000元人民币即可申请提现，机构在本月1-10号申请结算上月收入，双方确认无误后，机构提供与提现金额等值的增值税专用发票（抬头：北京无忧创想信息技术有限公司），51CTO在收到发票后即可完成银行转账（节假日顺延），转账账户为双方签订合作协议时提供的公司银行账户；如机构分成收入不足2000元，顺延到下月申请结算。
注意：只有机构的51CTO账户才能申请提现，机构讲师无提现权限。
机构上传课程（时）要求与规范
课程规范>>（请务必认真查看）
如何修改课程（时）信息？
成功发布课程后，进入"我的课程"页，点击相应课程的"编辑"即可修改该课程的相关信息。修改内容包括：课程标题、简介、课时数等等。
课程的定价标准？
只有互联网首发课程才能设置为高价付费课程，否则请各位讲师将课程设置为免费课程或者低价课程（低价课程建议10元内）。
培训套课或峰会会议视频：根据实际，按照整套课程价值定价，不受课时数限制，一般为线下定价的1/10-1/5。
如何为付费用户答疑？
为了提升付费用户的学习积极性以及给付费用户不一样的特殊待遇，51CTO学院规定讲师需要在24h之类对我们付费用户的问题给予答疑。
方法：每个课时页面都有"付费用户答疑专区"，付费用户可在此提问问题，讲师也会收到站内消息通知，届时在页面在线回答问题即可。
为什么要提交课件，如何提交？
为了学员更完善更系统的学习课程，请讲师上传您的课件。课件（PPT word pdf）只有付费的会员可以下载，方法：进入编辑课时页面，进行课件上传，如下图：
什么是自测题？何时提交？
自测题是各位讲师的付费课程所必须要提供的，放在相应课程的页面，目的是为了给我们的付费用户检测学习效果以增强用户对课程更高的认可度。
一套付费课程至少配套一套题目，由至少10个选择题构成，讲师也可根据课程的实际情况，出多套题目。 自测题目必须在课程上线后一个星期内提交给管理员。
何时签订合作协议及注意事项？
为了保护双方的利益，在合作时，51CTO会和机构签署合作协议。
签署时间：双方达成合作意向后即可签署协议。
注意事项：请机构在信息表中务必提供完整的信息（尤其是：地址、公司银行账户），以免增加双方的沟通成本。
合作协议签订流程？
51CTO通过邮件形式发PDF合作协议，讲师收到后打印黑色签字笔签字（一式两份），快递给51CTO（到付即可），51CTO盖章签字后寄给讲师。
