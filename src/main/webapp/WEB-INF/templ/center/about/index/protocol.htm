《51CTO学院在线培训协议》

 

尊敬的51CTO学院用户：

欢迎您使用51CTO学院在线学习服务。为保证您的学习服务权益得到保障，在开始付费购买程序之前，请您认真阅读以下条款。在您点击支付并开始学习过程时，就意味着您完全同意并接受以下服务条款。

 

1、本服务基本形式及要求

（1）本学习培训采用①线上直播②直播+录播③录播+作业考核方式进行在线学习服务。

（2）为保证学习效果，51CTO学院向您提供符合企业用人需求或考试认证要求的内容培养体系及相关课程，承诺配备技术讲师、班主任、辅导老师为您提供教学和学习服务（服务知识点仅限于为学员所报名课程的相关内容，超出课程范围以外的内容视讲师的繁忙程度而定，51CTO学院不提供承诺）。

（3）51CTO学院承诺在您学习期间，按照学员相关管理规定对您进行严格管理，并根据规定对您的表现进行评定考核，您同意接受考核。对于遵守并通过相关培训项目制作及考核的毕业学员，将颁发51CTO学院课程结业证书，并进入51CTO学院雇主人才库，同时针对有需要的同学进行就业推荐服务。对于考试认证类课程，51CTO学员承诺在学习结束后协助学员报名并完成考试，但与考试相关的所有相关费用不在51CTO学院服务之内。

（4）【特别提醒】学习是一个需要坚持的过程。为确保学习效果，您必须在规定时间内自助学习完录播视频培训内容，并准时参加直播上课、提交作业；如有特殊事宜，必须通过指定方式请假，征得班主任同意后方可不参加直播课。为保证学习效果，本学习过程您出勤率必须要求为70%以上，作业提交率必须为100%——未达到上述数字，将可能影响到您享有的其他服务（包括证书、推荐就业等），且51CTO学院不为此承担相应的任何赔偿责任。

（5） 为了保证学习效果，51CTO学院会为因自身学习能力不强或基础薄弱而未能达到合格成绩的学员（未获得51CTO学院结业证书），免费提供一次课程重听服务（限初次课程结束后半年内，且只能享受一次），且不再另行收取您任何学习费用；但享受免费重听服务必须满足前述两个条件：在初次培训过程中的出勤率不低于70%，且作业提交率为100%。

 

2、学习权限及退款

（1）一旦您用认可支付方式购买了51CTO学院在线培训服务，即视为您认可该项服务标明之价格；您购买成功后，该项服务即时生效。 如果您是通过贷款形式支付费用，那么您需按照贷款时的贷款协议按时还款；您和贷款提供方之间的所有服务内容，与51CTO学院无关。您应切实评估自己的还款能力，不得以无法向第三方还款为由解除本协议。

（2）学习时间权限：除特别标注外，本学习服务的有效期为2年，即自购买之日起2年内，录播课程可免费在线观看，但直播服务、答疑服务等非录播课程服务由于有比较强的时间性，所以仅限在指定的学习时间内，过期不补。

（3）51CTO学院为您提供完备的试学和退款服务，具体规则如下：

a）付款后的3个自然日（含）内支持无理由全额退款。但在付款和退款交易过程产生的支付给第三方的手续费应由您自行承担（例如信用卡POS机刷卡的交易手续费为乙方实际支付费用的3%）。如学员此前为贷款，退费时需自行承担贷款利息费，贷款利息费为贷款本金的10%。退款后本合同自行终止。

b）付款后超过3个自然日至指定期限，支持无理由退款50%。这里的“指定期限”指：若课程周期少于2个月，则为付款后3个自然日至14个自然日（含）内；若课程周期大于2个月，则为付款后3个自然日至1个自然月（含）内。退款时，51CTO学院已经交付您的材料需按材料（书籍、视频等）定价金额从需要退还给费用中扣除。如乙方为贷款学员，则乙方未还清给贷款公司的本金部分乙方须继续按照还款协议还款，直至还清。退款后本合同自行终止。

c）以下情况不接受退款：超过上述两款的退款约定时间不予退款；特价产品（包括且不限于“双11”、“周年庆”等特殊节假日活动的特价产品）不享受退款服务；任何形式的定金不予退还；直播课程的插班学员（开课后报名学员）不享受退款服务。

d）本协议所涉及的“退费”，以实际最终收到的学费为准，不包任何优惠活动所抵消的学习费用，如各种代金券、优惠券的费用。
